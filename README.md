# Cherry Shrimps Basics

A small educational website about cherry shrimps, created as a learning project using **plain HTML, CSS, and JavaScript**.

## About the project

This project focuses on building a clean, mobile-first informational website without frameworks.
It demonstrates semantic HTML, responsive CSS, basic accessibility practices, and minimal JavaScript for interaction.

Content and illustrations were generated with the help of AI tools and are intended for educational and demonstration purposes.

## Tech stack

* HTML
* CSS
* JavaScript

## Usage

The source code is free to use, modify, and learn from.
